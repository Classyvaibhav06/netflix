# Netflix Clone Design System

## Visual DNA
- **Theme:** Cinematic Dark Mode
- **Background:** High-contrast pure black (`#000000`)
- **Primary Color:** Netflix Red (`#E50914`)
- **Secondary Color:** Netflix Dark Grey (`#141414`)
- **Typography:** Sans-serif (Netflix Sans variant), bold headings.
- **Rounding:** 4px (Soft precision)
- **Effects:** Glassmorphism, card zoom on hover, subtle gradients.

## Screen Tokens
- **Hero:** Full-width high-res imagery with gradient overlays.
- **Card Grid:** 6-column responsive grid.
- **Overlays:** Semi-transparent dark surfaces with 20px blur.

## Components
- **Navbar:** Sticky, transparent to black transition.
- **Buttons:** 
  - Play: Solid white, black text.
  - Info: Semi-transparent grey, white text.
- **Cards:** Aspect ratio 2:3 (Posters) or 16:9 (Backdrops).
